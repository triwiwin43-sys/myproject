## PANDUAN IMPORT KE CANVA:

### LANGKAH-LANGKAH:
1. **Buka Canva.com**
2. **Pilih "Presentation"** template
3. **Gunakan "Text" tool** untuk setiap slide
4. **Copy-paste** konten dari setiap slide
5. **Tambahkan visual elements:**
   - Icons untuk bullet points
   - Charts untuk metrics
   - Screenshots untuk demo
   - Diagrams untuk arsitektur

### REKOMENDASI DESIGN:
- **Color Scheme:** Biru profesional (#1E40AF) + Putih
- **Font:** Montserrat atau Inter (clean & modern)
- **Layout:** Minimal dengan white space yang cukup
- **Visual Elements:** Icons dari Canva library
- **Consistency:** Template yang sama untuk semua slide

### TIPS PRESENTASI:
- **Durasi:** 20-25 menit + 5 menit Q&A
- **Slide Timing:** 1-2 menit per slide
- **Demo:** Siapkan video backup jika live demo gagal
- **Handout:** Print slide key points untuk penguji
