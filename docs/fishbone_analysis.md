# Fishbone Diagram - Analisis Masalah Inter Medi-A E-Commerce

## Problem Statement: Rendahnya Efisiensi Operasional E-Commerce

```
PEOPLE                    PROCESS                    TECHNOLOGY
   |                         |                          |
   |-- Manual inventory      |-- Tidak ada sistem      |-- Sistem tidak terintegrasi
   |-- Kurang training       |   tracking otomatis      |-- Database tidak sinkron
   |-- Customer service      |-- Proses checkout        |-- Tidak ada real-time update
       terbatas                  kompleks               |-- Mobile responsiveness kurang
   |                         |-- Konfirmasi payment     |
   |                             manual                 |
   |                         |                          |
   |                         |                          |
MATERIAL                  ENVIRONMENT               MEASUREMENT
   |                         |                          |
   |-- Stock tidak akurat    |-- Kompetitor online      |-- Tidak ada analytics
   |-- Supplier terbatas     |   semakin banyak         |-- KPI tidak terukur
   |-- Kualitas produk       |-- Customer expectation   |-- ROI tidak jelas
       tidak konsisten           tinggi                 |-- Conversion rate rendah
   |-- Packaging kurang      |-- Market trend berubah   |
       profesional               cepat                  |
```

### Penjelasan Masalah:

**1. People (SDM):**
- Inventory masih manual menyebabkan human error
- Staff kurang terlatih dalam sistem digital
- Customer service terbatas jam operasional

**2. Process (Proses):**
- Tidak ada sistem tracking otomatis untuk pesanan
- Proses checkout terlalu kompleks untuk customer
- Konfirmasi pembayaran masih manual

**3. Technology (Teknologi):**
- Sistem tidak terintegrasi antara inventory-sales-shipping
- Database tidak real-time sync
- Website kurang mobile-friendly

**4. Material (Material):**
- Data stock tidak akurat real-time
- Supplier terbatas untuk beberapa produk
- Packaging kurang profesional

**5. Environment (Lingkungan):**
- Kompetisi e-commerce semakin ketat
- Customer expectation tinggi untuk fast delivery
- Market trend berubah cepat

**6. Measurement (Pengukuran):**
- Tidak ada sistem analytics untuk track performance
- KPI bisnis tidak terukur dengan baik
- ROI dari marketing tidak jelas
