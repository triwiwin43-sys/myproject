# Strategi SEO - Inter Medi-A E-Commerce Platform

## 1. SEO On-Page Strategy

### A. Technical SEO
- **Site Speed Optimization**:
  - Compress images (WebP format)
  - Minify CSS/JS files
  - Enable Gzip compression
  - Use CDN for static assets
  - Target Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1

- **Mobile-First Indexing**:
  - Responsive design implementation
  - Mobile-friendly navigation
  - Touch-optimized buttons
  - Fast mobile loading speed

- **URL Structure**:
  - SEO-friendly URLs: `/printer/canon-pixma-g2010`
  - Breadcrumb navigation
  - XML sitemap generation
  - Robots.txt optimization

### B. Content Optimization
- **Title Tags** (50-60 characters):
  - Homepage: "Inter Medi-A | Printer, Komputer & Office Equipment Terbaik"
  - Category: "Printer Canon Terbaik | Harga Murah | Inter Medi-A"
  - Product: "Canon PIXMA G2010 | Printer Inkjet Multifungsi | Rp 2.1 Juta"

- **Meta Descriptions** (150-160 characters):
  - Compelling copy dengan call-to-action
  - Include target keywords naturally
  - Unique untuk setiap halaman

- **Header Tags (H1-H6)**:
  - H1: Main keyword (1 per page)
  - H2-H3: Supporting keywords dan struktur content
  - Logical hierarchy untuk readability

### C. Keyword Strategy
- **Primary Keywords**:
  - "printer murah Jakarta"
  - "komputer bekas berkualitas"
  - "service printer Canon"
  - "toner printer HP original"

- **Long-tail Keywords**:
  - "printer Canon PIXMA untuk UMKM"
  - "service printer di Jakarta Selatan"
  - "komputer untuk kantor harga terjangkau"

- **Local SEO Keywords**:
  - "toko printer Jakarta"
  - "service komputer Tangerang"
  - "printer murah Bekasi"

## 3. Social Media SEO Strategy

  - Regular posting schedule (3x/week)
  - Customer reviews dan testimonials
  - Facebook Shop integration

- **Content Strategy**:
  - Product showcase dengan specs
  - Behind-the-scenes content
  - Customer success stories
  - Live Q&A sessions

- **Facebook Ads**:
  - Lookalike audiences based on customers
  - Retargeting website visitors
  - Local area targeting (Jakarta, Tangerang, Bekasi)

### B. Instagram Strategy

- **Hashtag Strategy**:

## 5. E-commerce SEO Specifics

### A. Product Page Optimization
- **Product Descriptions**:
  - Unique content untuk setiap produk
  - Include specifications dan benefits
  - Use bullet points untuk readability
  - Add FAQ section

- **Image Optimization**:
  - Alt text dengan product keywords
  - Multiple angles dan zoom functionality
  - Image file names dengan keywords
  - Structured data markup

### B. Category Page SEO
- **Category Descriptions**:
  - Informative content above product grid
  - Include buying guides
  - Filter dan sort options
  - Pagination optimization

### C. Schema Markup
- **Product Schema**:
  - Price, availability, reviews
  - Brand, model, specifications
  - Shipping information

- **Organization Schema**:
  - Business contact information
  - Social media profiles
  - Logo dan address
