## ABSTRAK

**[NAMA MAHASISWA]. [NIM]. Pengembangan Sistem E-Commerce Inter Medi-A untuk Penjualan Printer, Komputer dan Peralatan Kantor Berbasis Web. Tugas Akhir. Program Studi [Nama Prodi]. Fakultas [Nama Fakultas]. Universitas [Nama Universitas]. [Tahun].**

Perkembangan teknologi informasi dan komunikasi telah mengubah cara berbisnis, termasuk dalam industri penjualan peralatan kantor. Inter Medi-A sebagai perusahaan yang bergerak di bidang penjualan printer, komputer, dan peralatan kantor membutuhkan sistem e-commerce untuk meningkatkan jangkauan pasar dan efisiensi operasional. Penelitian ini bertujuan untuk mengembangkan sistem e-commerce berbasis web yang dapat mengelola penjualan online, manajemen inventory, sistem pembayaran, dan layanan purna jual.

Metodologi pengembangan yang digunakan adalah Software Development Life Cycle (SDLC) dengan pendekatan waterfall. Tahapan penelitian meliputi analisis kebutuhan sistem, perancangan sistem menggunakan UML, implementasi menggunakan teknologi React.js untuk frontend dan Node.js dengan MongoDB untuk backend, serta pengujian sistem. Analisis sistem berjalan dilakukan menggunakan activity diagram dan fishbone analysis untuk mengidentifikasi permasalahan existing.

Hasil penelitian berupa sistem e-commerce yang memiliki fitur lengkap meliputi: katalog produk dengan sistem pencarian dan filter, keranjang belanja, sistem checkout dengan multiple payment gateway, manajemen pesanan real-time, sistem pengiriman dengan kalkulasi biaya berdasarkan kota tujuan, dan dashboard admin untuk monitoring bisnis. Sistem juga dilengkapi dengan fitur service request untuk layanan purna jual dan sistem loyalty program untuk customer retention.

Pengujian sistem menunjukkan bahwa aplikasi dapat berjalan dengan baik dan memenuhi kebutuhan fungsional yang telah ditetapkan. Sistem berhasil meningkatkan efisiensi proses bisnis dengan otomatisasi inventory management, order processing, dan customer service. Implementasi strategi SEO dan digital marketing terintegrasi dalam sistem mendukung peningkatan visibility online dan customer acquisition.

**Kata Kunci**: E-commerce, Sistem Informasi, Penjualan Online, Web Application, Digital Marketing

## ABSTRACT

**[STUDENT NAME]. [STUDENT ID]. Development of Inter Medi-A E-Commerce System for Printer, Computer and Office Equipment Sales Web-Based. Final Project. [Study Program]. [Faculty]. [University]. [Year].**

The development of information and communication technology has changed the way of doing business, including in the office equipment sales industry. Inter Medi-A as a company engaged in selling printers, computers, and office equipment requires an e-commerce system to increase market reach and operational efficiency. This research aims to develop a web-based e-commerce system that can manage online sales, inventory management, payment systems, and after-sales services.

The development methodology used is Software Development Life Cycle (SDLC) with waterfall approach. Research stages include system requirements analysis, system design using UML, implementation using React.js technology for frontend and Node.js with MongoDB for backend, and system testing. Current system analysis was conducted using activity diagrams and fishbone analysis to identify existing problems.

The research results in the form of an e-commerce system that has complete features including: product catalog with search and filter systems, shopping cart, checkout system with multiple payment gateways, real-time order management, shipping system with cost calculation based on destination city, and admin dashboard for business monitoring. The system is also equipped with service request features for after-sales service and loyalty program system for customer retention.

System testing shows that the application can run well and meet the functional requirements that have been set. The system successfully improves business process efficiency with automation of inventory management, order processing, and customer service. Implementation of SEO strategies and digital marketing integrated in the system supports increased online visibility and customer acquisition.

**Keywords**: E-commerce, Information System, Online Sales, Web Application, Digital Marketing
