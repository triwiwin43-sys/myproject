# 4.8 Pengujian Sistem

Pengujian sistem dilakukan menggunakan metode black box testing untuk memastikan setiap fungsi sistem berjalan sesuai dengan kebutuhan pengguna. Sistem Inter Medi-A E-Commerce telah di-deploy di platform Vercel dengan URL: **https://intermedia-store.vercel.app** dan telah melalui serangkaian pengujian komprehensif.

## 4.8.1 Metodologi Pengujian

### **Black Box Testing**
Pengujian dilakukan dengan fokus pada fungsionalitas sistem tanpa memperhatikan struktur internal kode. Setiap fitur diuji berdasarkan input yang diberikan dan output yang dihasilkan.

### **User Acceptance Testing (UAT)**
Pengujian dilakukan dengan melibatkan pengguna akhir untuk memastikan sistem memenuhi kebutuhan bisnis Inter Medi-A.

### **Cross-Browser Testing**
Sistem diuji pada berbagai browser untuk memastikan kompatibilitas:
- Google Chrome 120+
- Mozilla Firefox 119+
- Safari 17+
- Microsoft Edge 119+

### **Responsive Testing**
Pengujian dilakukan pada berbagai ukuran layar:
- Desktop (1920×1080px, 1366×768px)
- Tablet (768×1024px, 1024×768px)
- Mobile (375×667px, 414×896px)
